# Yōji

![App Icon](https://raw.githubusercontent.com/eleventytwo1172/Yoji/main/images/yoji-brand.png)  

[Check out the app](https://yoji-peach.vercel.app) or, [Download the app](https://github.com/eleventytwo1172/Yoji/releases)

Yōji in Japanese is "errands to attend to"

This project is a simple to-do list web app built with plain HTML, CSS, and JavaScript. It lets you create, manage, and delete daily tasks through a clean, minimal interface. The app saves everything locally in your browser, so your tasks stay even after refreshing or closing the tab. It’s fast, responsive, and easy to personalize for your own workflow.
